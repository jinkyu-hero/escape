마법서에 봉인된 당신! 네 개의 함수 관문을 통과하고 최종 좌표를 찾아 탈출하세요. 힌트는 이전 문제의 정답입니다.

[관문 1] 합성함수의 금고 (3자리 비밀번호)

함수 \(f(x) = 2x - 3\), \(g(x) = x^2 + 1\), \(h(x) = 5 - x\) 가 있습니다.

A: \(f(4)\)의 값

B: \((g \circ f)(3)\)의 값

C: \(\left|(f \circ h)(5)\right|\)의 값 (절댓값)

금고 비밀번호는 A, B, C를 순서대로 나열한 3자리 숫자입니다.

💡 개념 힌트: 합성함수 \((g \circ f)(x)\)는 \(g(f(x))\)임을 기억하고, 안쪽 함수 \(f(x)\)부터 먼저 계산하는 것을 잊지 마세요.

[관문 2] 역함수의 열쇠 (유리함수 \({k}\) 값)

함수 \(f(x) = \frac{1}{2}x + 4\) 가 주어졌습니다.

\(f^{-1}(7)\)의 값을 구하시오. 이 값이 다음 유리함수의 \({k}\) 값입니다.

💡 개념 힌트: \(f^{-1}(b)=a\)는 \(f(a)=b\)와 같습니다. 결과 값 \(7\)이 나오도록 하는 원래 함수 \(f(x)\)의 입력 값을 찾아보세요.

[관문 3] 유리함수의 키워드

이전 관문에서 얻은 \({k}\) 값을 이용하여, 함수 \(y = \frac{3x + k}{x - 2}\) 가 지나는 사분면의 개수를 구하시오.

(정답은 숫자만 입력)

💡 개념 힌트: 유리함수를 표준형 \(y = \frac{k'}{x-p} + q\)로 변형하고, 점근선의 교점 \((p, q)\)와 \(\mathbf{k'}\)의 부호를 확인하여 그래프를 그려보세요.

[최종 관문] 그래프 개형 코드 (M)

이전 관문 3의 정답을 상수항 \({k}\) 값으로 하는 유리함수 \(f(x) = \frac{{kx-7}{x-t} \)와 역함수가 일치할 때 \({t})\)의 값을 구하시오.

이 값이 최종 탈출 코드 \({t}\) 값입니다.

💡 개념 힌트: 유리함수의 점근선의 교점을 생각하자.
